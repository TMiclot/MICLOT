# Get structure properties from a PDB file

## 1. Sequence and Secondary structure

**Code**

```python
chainID_sequence, chainID_ss = get_sequence_secstruct('5azz.pdb')
print("chainID_sequence:", chainID_sequence)
print("chainID_ss:", chainID_ss)
```

**Code**

```python
pdb_file = '6f4j.pdb'

# use MDTraj chainID. Multiple ID can be in the list like: [0,1]
receptor = [0]
ligand = [2]

all_ASA, protein_region = get_protein_region(pdb_file, receptor, ligand)
print(all_ASA)
print(protein_region)
```
